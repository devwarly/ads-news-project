Sobre o Site ADS Notícias

Este site é um projeto desenvolvido para a disciplina de Engenharia de Software I do curso de Análise e Desenvolvimento de Sistemas. Nosso objetivo é criar um portal dinâmico para a comunidade acadêmica, exibindo notícias do curso, grade curricular e informações sobre eventos.

Equipe e Desenvolvimento

O desenvolvimento deste projeto foi realizado pela turma de ADS do 3º período. Buscamos aplicar as melhores práticas de desenvolvimento web, utilizando tecnologias modernas como HTML, CSS e JavaScript, juntamente com o framework Bootstrap para garantir um design responsivo e acessível.

Nossa Missão

Acreditamos que a informação é uma ferramenta poderosa. Nosso objetivo é manter os alunos e professores do curso de ADS sempre atualizados sobre as novidades, eventos e conquistas da nossa comunidade, promovendo a integração e o engajamento.
